# <a name="id-element"></a>Id, élément

Spécifie l’ID unique de votre complément Office comme GUID.

**Type de complément :** Application de contenu, de volet Office, de messagerie

## <a name="syntax"></a>Syntaxe

```XML
<Id>string</Id>
```
