# <a name="targetdialects-element"></a>Élément TargetDialects

Définit les langues régionales prises en charge par ce dictionnaire.

**Type de complément :** volet Office

## <a name="syntax"></a>Syntaxe

```XML
<TargetDialects>
   ...
</TargetDialects>
```
